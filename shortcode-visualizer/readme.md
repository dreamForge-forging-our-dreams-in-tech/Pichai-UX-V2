== Description ==
THis is a plugin that helps you visualize your Wordpress shortcodes.
With this plugin you can easily edit and preview how your shortcodes look on your website.

== Installation ==

Note: Plugin can be download from the dreamForge Pichai UX Repository: https://github.com/dreamForge-forging-our-dreams-in-tech/Pichai-UX-V2
Go to Plugins > Add Plugin > Upload Plugin > select shortcode-visualizer.zip > Install Now – succes: the plugin is installed!

== FAQ ==

= How do I use EXPERT shortcode visualizer? =

The visualizer only works with it's own shortcode (EXPERT shortcode visualizer) this means that any other shortcodes will just show up normally, if you use the plugins dedictated shortcode then you can preview any shortcode you select in the dropdown menu and modify it's attributes.

== Behind the scenes ==

= How does this plugin work? =

1. It scans all files installed in your wordpress directory for shortcodes and extracts all shortcodes and their attributes
2. The user adds the EXPERT shortcode visualizer shortcode the their editor and it asks the user to select a shortcode they want to render.
3. The shortcode looks through its database of scanned shortcodes and retrieves the HTML and associated attributes.
4. It renders the HTML from the selected shortcode and displays the list of associated atttributes
